## Features

- shopify restyle
- expo-router
- maps polyline
- Images thumbhash
- native feel touchables
- optimized assets
- google fonts
- shadowed bottom tabs
- jest unit-testing

## Installation

1. Clone the repository
2. npm i
3. npx expo start

## Build preview build locally:

android: `eas build --platform android --profile preview --local`
